# Основные понятия теории систем

Система - совокупность **элементов и связей**, образующая единое целое, предназначенная для достижения **определенной цели**.

Структура системы - это совокупность **элементов и связей**

Элемент системы - неделимое (с точки зрения **цели** описания системы)

**Системный** блок

### Подсистема
Подсистема - часть системы, обладающая всеми свойствами системы

### Связи
- Прямые
- Обратные

### Цели
Цель - это желаемый результат функционирования системы


## Решение основных задач теории систем
- Кибернетика
- Теория информации
- Теория игр
    - Функциональная конкуренция двух или более конкурирующих сил, с целью достижения максимального выигрыша и минимального проигрыша
- Топология


Теория систем является фундаментальной междисциплинарной наукой


Системотехника - научное управление систем людей, машин, финансов, и т.п.

Инженерная психология - анализ приспособленности людей/систем/машин для достижения максимальной эффективности при минимуме затрат

Функция - деятельность, внешнее проявление свойств какого-либо объекта в данной системе отношений

Свойства - внешнее проявление способа, которое получает знания об объекте

Состояние системы обеспечивает знание свойств системы в конкретный момент времени.



Изменение состояния системы, исходом которой является некий результат


Равновесие - способность системы сохранять равновесие при воздействии внешних возмущений


Устойчивость - способность системы возвращаться в состояние равновесия после воздействия внешних возмущений


Развитие - последовательное изменение состояния системы от некоторого фиксированного момента времени


Цель - желаемый результат деятельности, достижимый в пределах некоторого интервала времени


## Закономерности систем
### Целостность и обособленность
Если изменение в некоторой части системы вызывает изменения в других частях системы (и системы в целом),
то говорят, что система обладает свойством целостности.

### Коммуникативность
Любая система 
- с одной стороны является подсистемой более высокого уровня
- включает в себя подсистемы более низкого уровня



Иерархия - последовательная декомпозиция исходной системы на ряд уровней с установлением отношений подчиненностей нижележащих уровней к вышележащим

# Признаки систем

### Целостность, связанность, относительная независимость от среды и систем
Внешняя среда образована элементами, не входящими в систему, но оказывающая влияние на состояние системы.

Открытые системы - в которых возможен объем информации материей и энергией с внешней вредой


### Наличие подсистем и связей между ними (наличие структуры)
Связи разрушаются, система разрушается



### Возможость обособления (абстрагирования) от внешней среды
Относительная обособленность от тех факторов, которые мало влияют на систему


### Связь с окружающей средой по обмену ресурсами

### Подчиненность всей организации системы некоторой цели

### Эмерджентность
Несводимость свойств системы к свойствам элементов



Основная цель - удовлетворение информационных потребностей пользователя

Сложность поведения нелинейных характеристик


## Спецификация систем
- Формулируется цель (для чего?)
    - По видам научных направлений
        - Математические, биологичение, и т.п.
- По степени определенности фунционирования
    - Детерминированные
    - Вероятностные (стохастические)
- По степени организованности
    - Хорошо организованные
    - Плохо организованные (диффузные)
    - Самоорганизующиеся системы
- По происхождению
    - Естественные
    - Искусственные
- По взаимодействию со средой
    - Замкнутые
    - Открытые
- По степени сложности
    - Простые
    - Сложные
    - Очень сложные
- По естественному разделению
    - Технические
    - Биологические
    - Социально-экономические
- По выходным сигналам
    - Динамические
    - Статические
- По изменению во времени
    - Цифровые
    - Дискретные
- По типу организации
    - Централизованные
    - Децентрализованные


Объектом изучения системного анализа - стохастические, открытые, сложные и очень сложные системы любого происхождения


Хорошо организованные системы
- Солнечная система
- ...

Плохо организованные (диффузные) системы
- Теория вероятностей
- ...

Самоорганизующиеся системы
- Позволяет изучить  объекты и процессы
- Обладает свойствами диффузных систем
- Добавляется
    - Непредсказуемость системы
    - Приспособление к условиям окружающей среды
    - Изменять структуру взаимодействия
- Сохраняет свойство целостности

Самовосстанавливающиеся системы


### Большие и сложные системы
- По числу элементов
    - Малые
        - 10..1000 элементов
    - Сложные
        - 10^4...10^7
    - Ультрасложные
        - 10^7...10^30

### Сложные системы
Недостаточно информации для эффективного управления системой

От сложных систем необходимо отличать большие системы.
- Если ресурсов не хватает для системы, то эта система называется большой

### Особенности больших систем
- Большое число элементов в системе
- Иерархичность структуры управления
- Обязательное наличие человека в контуре управления

#### Примеры больших систем
- Производственный процесс
- Транспортная система в городе
- Кибернетическая система
- Система управления полетами
- ...

### Простая система
Только два состояния
- Исправное состояние
- Состояние отказа
