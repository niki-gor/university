# Что такое алгоритмы
Алгоритм - совокупность точно заданных правил решения некоторого класса задач или набор инструкций, описывающих порядок действий исполнителя для решения определенной задачи.

### Свойства алгоритмов
- Дискретность
- Детерминированность
- Понятность
- Завершаемость
- Массовость
- Результативность

### Способы описания алгоритмов
- Словесный
- Псевдокод
- Язык программирования
- Графически

```java
import java.util.Scanner;
public class FindMaximum {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.print("Введите первое число (а): ");
        int a = scanner.nextInt();
        System.out.print("Введите первое число (b): ");
        int b = scanner.nextInt();
        int maximum;
        if (a > b) {
            maximum = a;
        } else {
            maximum = b;
        }
        System.out.print(maximum);
    }
}
```
Максимальный элемент
```py
print(max([int(i) for i in input().split()]))
```
Затестить
```sh
echo '1 2 777 23' | python3 -c 'print(max(map(int, input().split())))'
```

### Основные алгоритмические конструкции (структуры)
Принцип структурного программирования - комбинируя ТРИ базовые структуры можно создать алгоритм для решения любой задачи

1. Следование (Linear Sequence)
    1. Команды выполняются строго последовательно, одна за другой, в том порядке, в котором они записаны
    2. Пример - взять зубную щетку - нанести пасту - почистить зубы - прополоскать рот
2. Ветвление (Selection/Condition)
    1. В зависимости от выполнения или невыполнения некоторого условия, алгоритм продолжается по одному из двух или более путей
    2. Ключевые слова - ЕСЛИ... ТОГДА... ИНАЧЕ...
3. Цикл (Repetition/Loop)
    - Некотороая последовательность команд повторяется несколько раз.
    - Существует два основных типа циклов:
        - Цикл с предусловием
        ```
        ПОКА (хлеб в тостере не подрумянен)
            жарить хлеб
        КОНЕЦ ЦИКЛА
        ```
        - Цикл с постусловием
        ```
        ДЕЛАТЬ
            звонить другу
        ПОКА (друг не берет трубку)
        ```
