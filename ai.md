Основная задача искусственного интеллекта являются выполнение вычислений без участия человека

1. Вычисление предоставляется в виде последовательности элементарных и рутинных операций
2. Формирование последовательности элементарных операций или составления алгоритма решения задачи предоставляется непосредственно пользователем вычислительной системыэ
3. Вычислительное устройство не может само ни создавать, ни менять алгоритм, если его изменение не предусмотрено самим алгоритмом


Данные - объекты, процессы, явления предметной области, а также их свойства
Знания - выявленные закономерности предметной области (принципы, законы, связи), позволяющие решать задачи в этой области

### Модели представления знаний
Модели представления знаний - это специальные языки представления знаний, основанные на символьном представлении данных.
1. Семантические сети
граф, вершины П1, П2, ..., ПN

Конкретные объекты и абстрактные объекты 

2. Фреймы
Фрейм - это единица представления знаний, заполненная в прошлом, детали которой по необходимости изменяются и уточняются применительно к ситуации.

- Скелетный, пустой фрейм (шаблон), который превращается после его заполнения в общее или конкретное понятие
- Фрейм общего понятия (прототип) - шаблон, который должен быть заполнен переменными
- Фрейм конкретного понятия (экземпляр) - прототип, который заполняется константами

3. Формальные логические модели

`M = <T, P, A, B>`

где 
- M - некая система 
- T - множество базовых элементов
- P - множество синтаксических правил
- A - множество элементов аксиом
- B - множество правил вывода

4. Продукционные модели
Продукционная модель - это модель, которая основана на практике. Позволяет представить знания в виде предложения

"Если (условие), то (действие)"

предложение-образец, по которому осуществляется поиск в базе знаний, а под действием - действие при успешном исходе поиска


# Языки программирования в системах искусственного интеллекта

Prolog базируется на ограниченном наборе механизмов, включающих в себя сопоставление образцов, древовидное представление структур данных и автоматический возврат

В самом названии Prolog есть сокращение, означающее "Programming Logic" ("ПРОграммирование в терминах ЛОГики")


Предикатом называется функция, возвращающая логическое значение (true или false) в зависимости от переданного ей значения.


Пример 1. Программа, содержащая факты

```prolog
родитель(пам, боб).
родитель(том, боб).
родитель(том, лиз).
родитель(боб, энн).
родитель(боб, пат).
родитель(пат, джим).
```
Запрос
```prolog
родитель(X,боб)
```
Ответ
```prolog
X = пам
X = том
```

---

Данная программа содержит 6 фактов, каждый объявляет 

В  Prolog важна последовательность утверждений

```prolog
спорт('бадминтон').
спорт('футбол').
спорт('теннис').
спорт('бейсбол').

любит('эллен', чтение).
любит('джим', бадминтон).
любит('эрик', чтение).
любит('марк', компьютеры).

спортсмен(X) :- спорт(S), любит(X,S).
```
Запрос
```prolog
спортсмен(X)
```
Ответ
```prolog
X = джим
```
